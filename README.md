# VirtualExpo

Небольшое веб-приложение с двухуровневой навигацией:

- коридор 3D-выставок, где пользователь выбирает зал по картинам;
- отдельные 3D-залы выставок с интерактивными объектами и медиаданными.

## Локальный запуск

```powershell
node server.js
```

После запуска откройте:

```text
http://127.0.0.1:4173
```
